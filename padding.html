Cuadro con Padding en CSS

Este es un ejemplo de cómo utilizar las propiedades de padding en CSS para crear espacio alrededor del contenido dentro de un elemento.

Explicación:
La propiedad padding establece el relleno alrededor del contenido del elemento.
Se puede especificar un valor de relleno para todos los lados a la vez o valores individuales para cada lado.
En este ejemplo, hemos establecido un relleno de 40px en todos los lados del cuadro.
Esto crea un espacio adicional entre el borde del cuadro y su contenido, proporcionando un aspecto más espaciado y agradable visualmente.
