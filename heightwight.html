Ejemplo de Tamaño de Elementos

Este es un ejemplo de cómo usar las propiedades de altura (height), anchura (width) y anchura máxima (max-width) en CSS para controlar las dimensiones de un elemento.
Explicación:
La propiedad height establece la altura del elemento.
La propiedad width establece la anchura del elemento.
La propiedad max-width establece la anchura máxima del elemento.
En este ejemplo, no hay una caja específica, sino que el contenido se muestra directamente. Además, el color del texto se ha cambiado para mejorar la legibilidad.
