Ejemplo de Propiedades de Fuente en CSS

Este es un ejemplo de cómo aplicar varias propiedades de fuente en CSS.
Font family: Cambia la fuente de letra.
Font style: Cambia el estilo de la fuente.
Text-decoration: Subraya el texto.
Text-transform: Transforma el texto a mayúsculas.
Letter-spacing: Espacia las letras del texto.
